# WA-010 - Deprecation burn-down: `action_view.raise_on_missing_translations`

## What changed
Rails deprecated `action_view.raise_on_missing_translations`.

Replaced it with the supported config key:

```ruby
config.i18n.raise_on_missing_translations = true
```

Also updated commented-out development environment examples to match the new key.
